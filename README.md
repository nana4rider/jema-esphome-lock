# JEM-A ESPHome Lock

## 概要

JEM1427(HA端子、JEM-A端子)をGPIOに接続し、Home Assistantで操作するためのドキュメントです。

Raspberry Pi Zero 2 Wに下記の回路を取り付けることで動作します。
